CSS Media Queries

How does a CSS media query work, and why is it useful?

CSS media queries take information from a device, like screen orientation or media type, and use breakpoints in your CSS code to make changes to design elements in response to that information. Once you add a media query breakpoint, like device orientation, you can then specify a design element that will apply when the device meets that condition.

What "media features" are supported by media queries-- such as "orientation" or "max width"-- and what do they do?

CSS media queries support media features such as device width and height, viewport minimums and maximums, aspect ratio, device orientation, resolution, and more.

The media feature "max-width" adjusts content based on the width of the device. This is useful for scaling content to be compatible with mobile devices, tablets, laptops, desktops, etc.
