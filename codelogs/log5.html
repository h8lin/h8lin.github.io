What are the differences between presenting an image using an HTML tag (like img) and presenting it as the background of an HTML element (using the CSS property background-image)?

When adding images with and HTML tag, that image becomes an element on the page. When using the property background-image, the image becomes the background for an element and is then contained within the element.

In one sentence, how would you define the differences between dimensions and resolution in a bitmap image?

In bitmap images, the dimensions describe the size of the image in width x height, and the resolution describes the number of pixels in a certain area, usually pixels per inch (PPI) and dots per inch (DPI).
